# Bối Cảnh Dự Án

Kho lưu trữ này là Agent Skill Framework để xây dựng và duy trì các AI skill.

## Bắt Buộc Đọc
- Đọc `architect.md` trước mọi tác vụ.
- Xem `architect.md` là nguồn sự thật duy nhất về kiến trúc, workflow và zones.

## Phạm Vi
- Chỉ làm việc với skill assets và tài liệu liên quan trong các zone hợp lệ.
- Không triển khai tính năng sản phẩm hoặc đổi phạm vi khi chưa được chấp thuận.
- Mỗi skill phải tự hoạt động độc lập. Nếu cần tham chiếu, chỉ dùng nguồn internet hoặc file trong thư mục skill. Cần copy thông tin vào thư mục skill thay vì tham chiếu cứng tới file cấp dự án.

## Tiêu Chí Thành Công
- Output bám `architect.md`.
- Tuân thủ các phase và verify checkpoint.
