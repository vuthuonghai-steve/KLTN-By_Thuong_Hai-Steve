# Mẫu Cấm

## Tuyệt Đối Không Làm
- Đề xuất thay đổi kiến trúc trái với `architect.md` khi chưa được phê duyệt.
- Tạo file ngoài zone hợp lệ khi chưa có xác nhận người dùng.
- Bỏ qua bất kỳ phase nào hoặc verify checkpoint.
- Chuyển phạm vi từ làm skill sang viết mã sản phẩm khi chưa xác nhận.
- Che giấu lỗi hoặc tiếp tục sau khi verify thất bại.
- Dùng quy tắc đặc thù nền tảng làm mất tính tương thích giữa các AI agents.
- Phụ thuộc file cấp dự án như tài nguyên chạy của skill. Skill phải tự chứa tài liệu và chỉ dùng nguồn internet hoặc file trong thư mục skill.
