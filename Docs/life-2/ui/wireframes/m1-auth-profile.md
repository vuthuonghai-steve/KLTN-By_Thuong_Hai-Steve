# Wireframes: Module M1 - Authentication & Profile

> **Mục đích:** Phác thảo cấu trúc giao diện cho các màn hình Auth và Profile.  
> **Phong cách:** Pink Petals Design System (Sạch sẽ, Tối giản, Màu hồng chủ đạo).

---
