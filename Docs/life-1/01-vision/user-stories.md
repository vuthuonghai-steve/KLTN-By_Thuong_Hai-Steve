# User Stories

> **Mục đích:** Epic + User Stories cho 10 MVP features
> **Format:** As a [user], I want [feature], so that [benefit]

---

## Epic 1: Authentication

- **US-1.1:** As a user, I want đăng ký bằng email/password để tạo tài khoản
- **US-1.3:** As a user, I want đăng xuất để bảo mật khi dùng chung thiết bị

## Epic 2: Profile Management

- **US-2.1:** As a user, I want upload avatar và cập nhật bio để thể hiện bản thân
- **US-2.2:** As a user, I want thêm link mạng xã hội (GitHub, LinkedIn) vào profile
- **US-2.3:** As a user, I want xem profile người khác và danh sách bài viết của họ

## Epic 3: Content Creation (Posts)

- **US-3.1:** As a user, I want tạo bài viết text + image + link để chia sẻ
- **US-3.2:** As a user, I want thêm tags cho bài viết để dễ tìm kiếm
- **US-3.3:** As a user, I want sửa/xóa bài viết của mình

## Epic 4: Social Bookmarking

- **US-4.1:** As a user, I want lưu bài viết vào collection để đọc sau
- **US-4.2:** As a user, I want tạo nhiều collections (VD: React, Database) để tổ chức
- **US-4.3:** As a user, I want xem danh sách bookmarks và bỏ lưu

## Epic 5: News Feed

- **US-5.1:** As a user, I want xem feed bài viết từ người mình follow, sắp xếp theo ranking
- **US-5.2:** As a user, I want phân trang (cursor/pagination) khi scroll
- **US-5.3:** As a user, I want feed ưu tiên nội dung hot (engagement + recency)

## Epic 6: Interactions (Like, Comment, Share)

- **US-6.1:** As a user, I want like/unlike bài viết
- **US-6.2:** As a user, I want comment và reply nested
- **US-6.3:** As a user, I want share bài viết (copy link hoặc repost)

## Epic 7: Search

- **US-7.1:** As a user, I want tìm kiếm users, posts, tags
- **US-7.2:** As a user, I want autocomplete khi gõ search
- **US-7.3:** As a user, I want filter theo type (users/posts/tags)

## Epic 8: Notifications

- **US-8.1:** As a user, I want nhận thông báo realtime (SSE) khi có like, comment, follow
- **US-8.2:** As a user, I want đánh dấu đã đọc và xem danh sách notifications

## Epic 9: Moderation

- **US-9.1:** As a user, I want report bài viết/comment vi phạm
- **US-9.2:** As admin, I want xem reports và xử lý (xóa, cảnh cáo)
- **US-9.3:** Bài mới auto-approve, chỉ review khi có report

## Epic 10: Privacy & Connections

- **US-10.1:** As a user, I want follow/unfollow người khác
- **US-10.2:** As a user, I want block user
- **US-10.3:** As a user, I want cài đặt visibility profile (public/followers only)
